étape 1

  • Faire une demande sur ce site

    à l'étape 10 on vous demandera de vous inscrire pour validé votre demande, suite à cela, vous retrouverez votre numéro de demande, retenez le !

    • Si vous avez perdu le numéro de la demande, connecter vous avec votre identifiant ici pour le récupérer

      • Ce site vous permet également de suivre l'avancement de votre demande, tout les messages que vous recevrer est consultable ici.

étape 2

Une fois que vous avez votre numéro de demande, rendez-vous sur : Arrima pour payer le CAQ (128$can soit environ 87.32€).
Il vous faudra crée un compte à nouveau.

Une fois que votre compte est crée, rendez-vous dans la section Mes paiements puis rentrer votre numéro de demande.

étape 3

Une fois que vous avez payé les frais, vous devez attendre 24 à 48 heures le temps que le numéro du dossier à apparaisse sur le site du dossier

une fois que le numéro est apparut, retourner sur Arrima pour complété la partie mes dossiers.
